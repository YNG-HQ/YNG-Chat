# Contributing aan YNG

Bedankt dat je de documentatie bekijkt. Wil je bijdragen? Top!

## Code Stijl
- **Taal:** TypeScript.
- **Styling:** Tailwind CSS.
- **Componenten:** React.

## Workflow
1. **Branching:** Gebruik `feature/naam-van-functie` of `fix/naam-van-bug`.
2. **Commits:** Gebruik [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat:` voor nieuwe functies
   - `fix:` voor bugfixes
   - `docs:` voor wijzigingen in documentatie
3. **Pull Requests:** Houd ze klein en gefocust op één doel.

## Lokale Setup
1. `git clone https://github.com/yng-hq/yng-chat.git`
2. `npm install`
3. `npm run dev`
